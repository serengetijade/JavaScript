Expressions & Statements:

In computers, an expression is a combination of values that are computed by the computer. There are different ways to write out expressions depending on which language you are programming in. For example: Name = “What the person types in the name box” could be an expression. Also, x + 5 is an expression. In JavaScript, an expression results in a value. There are basically two types of expressions: ones that have a value (or result in a value) and ones that assign a value to a variable.
Statements are computer instructions. In JavaScript, a statement performs a task – a program is basically a series of statements. These are the instructions that are used by people as they create computer programs. The simplest of these might be things like “print,” “delete,” “add,” “multiply,” etc. A statement may contain an expression. Statements are separated by semicolons ;.
