break & continue Statments

The break statement "jumps out" of a loop. It breaks out of the switch block. This will stop the execution of more execution of code and/or case testing inside the block. If break is omitted, the next code block in the switch statement is executed.
The continue statement "jumps over" one iteration in the loop.
